VersionGuard feeds data into the ThreatLabs SaaS platform, designed for security teams and administrators managing multiple WordPress installations.

== Description ==
ThreatLabs offers a remote portal with a centralized dashboard for insights into the status of multiple WordPress installations. This SaaS facilitates the identification of outdated WordPress versions, plugins, and themes. Additionally, it offers support for automated alerts and notifications, ensuring you remain promptly informed about potential security issues. Please note that this is a paid service.
This plugin periodically collects and sends version information to a third-party API.

API terms of use and privacy policy: https://portal.threatlabs.eu/terms

== Installation ==
1. Install the plugin
2. Register the ID visible in the admin section in the ThreatLabs SaaS portal
